# docs 项目

本节将介绍如何开发、维护 **docs** 项目。

## 一、开发环境启动

::: tip 当前运行版本

- node v18.17.1
  :::

1. 安装依赖 `pnpm install`
2. 启动 **docs** 项目 dev server:
  - 项目根路径执行 `pnpm dev:docs`
  - 进入 **docs** 项目执行 `pnpm dev`
3. 启动 **本次开发的组件库** dev server:
  - 项目根路径执行 `pnpm dev`
  - 进入 **cxz-ui** 项目执行 `pnpm dev`
